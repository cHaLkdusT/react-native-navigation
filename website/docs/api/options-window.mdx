Provides a way to configure the overall presentation of your application's main user interface.

```js
const options = {
  window: {
    backgroundColor,
  },
};
```

### `backgroundColor`

Configure the background color of the application's main window.

| Type  | Required | Platform |
| ----- | -------- | -------- |
| color | No       | iOS      |
